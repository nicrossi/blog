This project implements a **Metropolis-Monte Carlo simulation** to model opinion dynamics on a 2D lattice grid,
where binary-state agents evolve through neighbor influence and probabilistic noise.
The system computes phase transitions by analyzing consensus formation, susceptibility metrics, and stationary
behavior across varying grid sizes and flip probabilities.

*Built for the Simulation Systems course at Buenos Aires Institute of Technology (ITBA).*
