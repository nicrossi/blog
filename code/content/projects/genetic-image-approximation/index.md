A custom-built genetic algorithm that evolves populations of semi-transparent
triangles to reconstruct target images with stunning artistic fidelity. Each
genome encodes triangle vertices, RGBA channels, and z-indices, while a pluggable
strategy architecture (selection, crossover, mutation) optimizes pixel-level
fitness across generationsâ€”producing compact, human-legible "compressed" artwork
from any source image.
