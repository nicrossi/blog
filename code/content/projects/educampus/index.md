A full-stack educational management platform built on a clean multi-layered architecture following
Spring MVC patterns with modular Maven design. The system implements JWT-based authentication,
RESTful APIs with JAX-RS, and a reactive TypeScript frontend, leveraging Hibernate ORM with
Flyway migrations for robust database management.

*Built for the Web Applications Project course at Buenos Aires Institute of Technology (ITBA).*
